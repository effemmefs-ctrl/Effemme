⚙️ Configurar Supabase

O Snapost usa o Supabase como banco de dados e autenticação. É gratuito e leva 2 minutos para configurar.

1
Acesse supabase.com e crie uma conta grátis. Clique em "New project".
2
No seu projeto, vá em Settings → API. Copie a Project URL e a anon key.
3
Cole abaixo. O Supabase cria a tabela de usuários automaticamente com o sistema de Auth embutido.
Suas credenciais ficam salvas apenas neste navegador.
Precisa de ajuda? Veja a documentação.
